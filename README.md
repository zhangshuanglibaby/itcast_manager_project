在组件加载完毕就要获取数据--使用mounted钩子函数

在@/utils/myaxios.js中使用拦截器设置请求报文,先从本地储存获取token,请求头添加Authorization=token,才能获取数据

在分页的时候,记得要更改参数中的数据

- 当切换当前页码触发时
- 当切换当前页码所显示的数量时

用户的数据搜索,其实也是调用了获取数据的请求,此时可以把获取数据的接口请求封装为一个函数.一个细节是,当现实错误信息是无效token的时候,让页面跳转到登录页面

实现输入的时候就能进行搜索,使用input事件,由于input是原生事件,需要使用native为文本框组件添加原生事件
